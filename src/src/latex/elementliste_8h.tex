\doxysubsection{Detailed Description}
Header file defining operations and types for list elements, specifically for \textquotesingle{}\mbox{\hyperlink{structt__arc__p}{t\+\_\+arc\+\_\+p}}\textquotesingle{} type elements. 

Ce fichier définit des opérations et des types pour les éléments de liste, plus particulièrement pour les éléments de type \textquotesingle{}\mbox{\hyperlink{structt__arc__p}{t\+\_\+arc\+\_\+p}}\textquotesingle{}. Il fournit des fonctions pour la saisie, la copie, l\textquotesingle{}affichage et la comparaison des éléments de ce type.
