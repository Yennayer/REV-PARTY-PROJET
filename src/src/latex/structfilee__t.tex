\doxysubsection{Detailed Description}
Structure pour gérer les informations relatives à un fichier de vote. 

Cette structure stocke les détails d\textquotesingle{}un fichier de vote, y compris le module de vote utilisé, le nom du fichier de vote, et un booléen indiquant si un fichier de log est présent.
