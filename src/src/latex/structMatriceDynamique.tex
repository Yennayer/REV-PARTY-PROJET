\doxysubsection{Detailed Description}
Structure représentant une matrice dynamique pour les duels de candidats. 

Cette structure stocke les informations relatives aux duels entre candidats, y compris le nombre de candidats, le nombre de duels, et une matrice tridimensionnelle pour représenter les résultats de ces duels.
