\doxysubsection{Detailed Description}
Header file defining operations and structures for handling a static list of \textquotesingle{}Elementliste\textquotesingle{} type elements. 

Ce fichier définit des opérations et des structures pour gérer une liste statique d\textquotesingle{}éléments de type \textquotesingle{}Elementliste\textquotesingle{}. Il comprend des fonctions pour la création, l\textquotesingle{}ajout, la suppression, et d\textquotesingle{}autres opérations sur la liste.
